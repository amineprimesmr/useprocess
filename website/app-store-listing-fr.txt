--- FACE DATA / TRUEDEPTH — COPY-PASTE REPLY TO APPLE (English) ---

1) What face data does the app collect?

Process AI collects face-related data ONLY when the user explicitly starts the optional "Face Scan" feature (TrueDepth / front camera). We do NOT use face data for identity recognition, authentication, or unlocking the device.

Data collected:
• 3D face mesh geometry (vertices, triangle indices) — processed on-device from TrueDepth
• JPEG snapshot of the face during the scan
• Optional short local video recording of the scan session
• Derived wellness scores (0–100): puffiness, under-eye fatigue, jaw tension, skin clarity, facial symmetry
• Scan metadata: date/time, optional sleep/HRV context from Apple Health if authorized
• Optional AI-generated text analysis (only if user enables "AI analysis of my photo")

The scan is optional. Users can use the app without ever running a face scan.

2) Planned uses of collected face data

• Personal wellness tracking and trend comparison over time (fatigue, recovery, debloat indicators)
• Display in the Health dashboard and coach context (scores and trends; not photos by default)
• Sync scan history across the user's devices when signed in
• Optional: send the JPEG snapshot + derived scores to Anthropic Claude (via our Firebase Cloud Functions proxy) to generate personalized wellness coaching text — ONLY if the user explicitly enables the in-app toggle on the Face Scan consent screen

We do NOT use face data for: identity verification, advertising, tracking, biometric login (Face ID login is separate Apple system auth and is not stored by us), or training third-party AI models.

3) Third-party sharing and storage

On-device (iPhone/iPad):
• Photos/videos: Application Support/FaceScans/ folder on the device
• Mesh and markers: local app storage (UserDefaults / Application Support)

Cloud (only if user is signed in):
• Firebase Firestore (Google): users/{userId}/faceScans — stores wellness scores, metadata, optional AI text analysis. JPEG filenames and video files are NOT uploaded to Firestore (stripped before sync). Max 90 scans retained per account (oldest auto-deleted).

Third parties:
• Anthropic (Claude): ONLY if user opts in on consent screen — JPEG snapshot + wellness scores sent via Firebase Cloud Functions for one-time analysis per scan. Anthropic API data is not used to train models per their commercial API terms.
• NOT shared with advertisers or data brokers. NOT used for tracking (NSPrivacyTracking = false).

4) Retention period

• Local + cloud scan history: maximum 90 scans per account; older scans deleted automatically
• Account deletion: all face data deleted within 30 days of account deletion request
• Anthropic: photo processed for the API request; retention governed by Anthropic's API privacy policy; we store only the returned text analysis with the scan record
