Politique de confidentialité

Dernière mise à jour : 21 juin 2026

La présente politique décrit comment Process AI (« l'Application », « nous ») collecte, utilise, partage et conserve vos données lorsque vous utilisez l'application mobile iOS disponible sur l'App Store.

1. Responsable du traitement

Éditeur : Process
Contact : support@useprocess.xyz
Site : useprocess.xyz

2. Données collectées

Selon les fonctionnalités que vous utilisez, nous pouvons traiter :

  • Compte : identifiant Apple (Sign in with Apple), adresse e-mail relayée le cas échéant, prénom.
  • Profil : objectifs, préférences sport et nutrition, réponses d'onboarding.
  • Données santé (avec votre autorisation HealthKit) : activité, sommeil, fréquence cardiaque, calories, pas, etc.
  • Données faciales (avec votre autorisation — voir section 3) : mesh 3D TrueDepth, photo, vidéo courte, scores wellness dérivés.
  • Activité physique (capteurs de mouvement, si autorisés).
  • Coach IA : messages échangés avec l'assistant (voir section 4).
  • Audio (si autorisé) : dictée vocale convertie en texte sur l'appareil.
  • Abonnement : statut d'abonnement via Apple et RevenueCat (aucune donnée bancaire n'est collectée par nous).
  • Technique : identifiants de session, logs d'erreur, version de l'app.

Localisation GPS : Process AI ne collecte pas votre position géographique. Aucune donnée de localisation n'est demandée ni transmise par l'application.

3. Données faciales (TrueDepth)

Si vous choisissez d'utiliser le scan visage, l'Application utilise la caméra TrueDepth (capteur Face ID) pour estimer des indicateurs wellness. Aucune reconnaissance biométrique d'identité (déverrouillage, authentification biométrique du visage) n'est effectuée dans le cadre du scan wellness.

3.1 Données faciales collectées

  • Mesh 3D facial : géométrie du visage (vertices, triangles), traitée principalement sur l'appareil.
  • Photo JPEG : capture d'image du visage pendant le scan.
  • Vidéo courte : enregistrement local du scan (stocké uniquement sur l'appareil, non téléversé).
  • Scores wellness dérivés : gonflement, cernes/fatigue, tension mâchoire, clarté peau, symétrie (échelle 0–100).
  • Métadonnées : date du scan, scores, sommeil/HRV au moment du scan si HealthKit est autorisé.

3.2 Utilisations

  • Suivi wellness personnalisé et comparaison de tendances dans le temps.
  • Synchronisation entre vos appareils via votre compte.
  • Enrichissement du contexte du coach (scores, pas la photo, sauf si vous autorisez l'analyse IA photo — voir 3.3).
  • Analyse IA optionnelle : si vous activez explicitement l'option dans l'Application, la photo du scan peut être envoyée à Anthropic (Claude) pour générer un texte d'analyse wellness.

3.3 Partage et stockage

  • Sur votre iPhone : dossier sécurisé de l'Application (Application Support).
  • Cloud : si vous êtes connecté, scores wellness et métadonnées synchronisés sur Google Firebase Firestore (collection faceScans), rattachés à votre identifiant de compte. Les fichiers photo JPEG et vidéo ne sont pas téléversés — ils restent sur votre appareil.
  • Anthropic (Claude) : uniquement si vous avez coché l'option d'analyse IA du scan — la photo JPEG et les scores associés sont transmis via nos Firebase Cloud Functions (proxy). Anthropic traite ces données pour produire l'analyse textuelle demandée. Selon les conditions commerciales d'Anthropic, les données API ne sont pas utilisées pour entraîner leurs modèles.

3.4 Durée de conservation

  • Historique local et cloud : jusqu'à 90 scans par compte (les plus anciens sont supprimés automatiquement, y compris sur Firebase Firestore).
  • Photos et vidéos locales : supprimées lorsque le scan sort de l'historique (90) ou lorsque vous révoquez le consentement scan visage dans les paramètres.
  • Suppression de compte : toutes les données faciales associées au compte sont supprimées sous 30 jours après votre demande.
  • Révocation du consentement : via Paramètres → Confidentialité → « Révoquer le scan visage » — supprime l'historique, les médias locaux et les données cloud associées.
  • Analyse IA : le texte d'analyse est stocké avec le scan ; la photo transmise à Anthropic est traitée pour la requête en cours (politique de rétention d'Anthropic applicable).
Extrait — collecte et usage des données faciales : « Si vous choisissez d'utiliser le scan visage, l'Application collecte un mesh 3D TrueDepth, une photo JPEG, une vidéo courte optionnelle et des scores wellness dérivés. Les photos et vidéos restent sur votre appareil ; si vous êtes connecté, les scores et métadonnées sont synchronisés sur Firebase Firestore. Si vous autorisez explicitement l'analyse IA, la photo peut être envoyée à Anthropic (Claude) via Firebase Cloud Functions. Aucune reconnaissance d'identité n'est effectuée. Conservation : max. 90 scans ; suppression sous 30 jours après suppression du compte. »

4. Intelligence artificielle tierce (Anthropic Claude)

L'Application intègre un coach et des fonctionnalités IA. Avant tout envoi de données personnelles à un service IA tiers, l'Application vous demande votre consentement explicite (écran dédié « Coach IA »). Vous pouvez refuser ou révoquer ce consentement dans les paramètres.

4.1 Prestataire

Anthropic PBC — modèle Claude (IA conversationnelle et vision).
Transmission via Google Firebase Cloud Functions (proxy sécurisé ; clé API côté serveur).
Politique Anthropic : anthropic.com/privacy

4.2 Données pouvant être envoyées (avec votre consentement)

  • Messages écrits ou dictés au coach et historique de conversation.
  • Profil : prénom, âge, genre, objectifs, sport, nutrition, réponses d'onboarding.
  • Résumés HealthKit : pas, sommeil, fréquence cardiaque, HRV, score readiness.
  • Scores de scans (visage, posture) et historique agrégé.
  • Photos que vous envoyez volontairement (coach, repas, scan visage si option IA activée).

4.3 Finalités

  • Réponses personnalisées du coach.
  • Génération et résumé de protocole.
  • Suggestions de repas et analyse d'images alimentaires.
  • Analyses wellness optionnelles (scan visage/corps).

4.4 Protection équivalente

Nous sélectionnons Anthropic en tant que sous-traitant offrant des garanties contractuelles et techniques appropriées (chiffrement en transit, conditions commerciales API, non-utilisation des données API pour l'entraînement des modèles selon leurs conditions). Des clauses contractuelles types s'appliquent pour les transferts hors UE le cas échéant.

Extrait — IA tierce : « Avec votre consentement explicite, Process envoie certaines données personnelles à Anthropic (Claude) via Firebase Cloud Functions pour le coach IA, les suggestions repas et les analyses optionnelles. Vous pouvez refuser ou révoquer à tout moment dans l'Application. Anthropic ne reçoit vos données qu'après votre accord in-app. »

5. Finalités générales

  • Fournir et personnaliser l'Application (programme, coach, suivi santé).
  • Synchroniser vos données entre appareils via votre compte.
  • Gérer les abonnements et restaurer vos achats.
  • Envoyer des rappels (notifications) si vous les activez.
  • Améliorer la fiabilité et la sécurité du service.
  • Respecter nos obligations légales.

6. Bases légales (RGPD)

  • Exécution du contrat : fourniture de l'Application et de l'abonnement.
  • Consentement : HealthKit, caméra/TrueDepth, notifications, micro, coach IA Anthropic, analyse IA photo du scan visage.
  • Intérêt légitime : sécurité, prévention de la fraude, amélioration du service.

7. Sous-traitants et services tiers

  • Apple : App Store, Sign in with Apple, HealthKit, achats intégrés, Speech (dictée on-device).
  • Google Firebase : authentification, Firestore, Cloud Functions (proxy IA).
  • RevenueCat : gestion des abonnements.
  • Anthropic (Claude) : coach IA et analyses vision — uniquement avec consentement explicite (section 4).

8. Durée de conservation

  • Données de compte : tant que le compte est actif, puis suppression sous 30 jours après demande.
  • Données santé synchronisées : supprimées avec le compte.
  • Données faciales : voir section 3.4 (max. 90 scans).
  • Conversations coach : conservées tant que le compte est actif ; supprimables par vous.
  • Données d'abonnement : selon obligations comptables et légales applicables.

9. Vos droits

Conformément au RGPD, vous disposez des droits d'accès, de rectification, d'effacement, de limitation, d'opposition et de portabilité. Vous pouvez retirer votre consentement (HealthKit, IA, scan visage) via les réglages iOS ou dans l'Application (Paramètres → Confidentialité IA).

Pour exercer vos droits : support@useprocess.xyz. Vous pouvez également introduire une réclamation auprès de la CNIL.

10. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles raisonnables (chiffrement en transit TLS, authentification Firebase, accès restreint) pour protéger vos données.

11. Transferts hors UE

Certains prestataires (Firebase, Anthropic) peuvent traiter des données aux États-Unis. Des garanties appropriées sont mises en place (clauses contractuelles types, etc.).

12. Mineurs

L'Application s'adresse aux personnes de 16 ans et plus. Si vous pensez qu'un mineur nous a transmis des données sans autorisation parentale, contactez-nous pour suppression.

13. Modifications

Nous pouvons mettre à jour cette politique. La date en tête de page sera modifiée en conséquence. En cas de changement substantiel, nous pourrons vous demander un nouveau consentement in-app.
