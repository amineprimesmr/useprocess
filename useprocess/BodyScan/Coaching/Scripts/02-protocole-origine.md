# Script 02 — Comment devenir beau de la tête aux pieds (Protocole Origine)

**Chaîne:** Enzo Geromegnace  
**Durée:** ~50 min  
**Thème:** système complet transformation — 4 piliers

## Concepts clés extraits

### Philosophie
- Pas la génétique → les habitudes (<20 % potentiel exploité)
- Indicateurs santé = beauté/fertilité/longévité (programmation biologique)
- Épigénétique > génétique
- Beauté = conséquence quand le système fonctionne
- 10 % des actions = 90 % des résultats — bon ordre, pas sauter d'étapes

### Pilier 1 — Hormones + système nerveux
- Apparence régie par hormones ; système nerveux envoie les signaux
- Toile d'araignée : une hormone bougée = tout bouge (pas TRT/pilule comme fix)
- Xénoestrogènes (plastique chauffé), œstrogènes/progestérone, testostérone
- 95 % en mode survie → pas de réparation
- Cortisol chronique vs aigu ; sympathique/parasympathique
- Réduire toxines environnement (priorité corps = éliminer toxines)
- Rythme circadien : lumière naturelle matin, pas bleu artificiel soir
- Mélatonine vs cortisol le soir
- Alimentation dense : animaux, fruits, miel — peu de toxines
- Soleil = pilier #1
- Sommeil profond = réparation hormonale

### Pilier 2 — Entraînement adapté
- Muscle se construit en réparation (système nerveux stable)
- Concurrent training / cycles 2-4 semaines
- Homme : 3-4×/sem, clavicules/trapèzes/épaules/cou/fessiers, dos en V
- Femme : 1-2×/sem intensité, cycle menstruel, fessiers/hanches
- Fonctionnel > machines (fascias)
- Progresser à l'entraînement = adaptation corporelle

### Pilier 3 — Posture + fascias
- Fascias = toile corps entier ; rigidité → lymph stagnant → cernes, fatigue
- Émotions cristallisées dans fascias → cercle vicieux avec posture
- Cage thoracique affaissée → diaphragme → respiration → hormones
- Tête en avant → langue quitte palais → maxillaire s'affaisse
- Pieds = fondations (chaussures étroites, marcher pied nu, démarche)
- Connexion pied-fessier
- Détecter déséquilibres AVANT protocole au hasard
- Libération fascias + renforcement zones sous-actives

### Pilier 4 — Résultats (conséquence)
- Peau, cheveux, regard ancré, structure faciale, silhouette
- Regard tiré = stress/cortisol/tensions front — pas chance
- Corps + esprit liés (testostérone ↔ action/confiance)

### Lien scan useprocess
- Asymétries/posture → pilier 3 + fascias
- Fatigue visage → pilier 1 (sommeil, cortisol, lymph)
- Score bas → bases hormones/posture avant tout hack
- Bassin → fessiers, pieds
