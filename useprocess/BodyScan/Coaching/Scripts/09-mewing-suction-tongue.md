# Script 09 — Mewing / suction mew (Oscar Patel)

**Thème:** langue = moteur croissance faciale, suction mew 24/7, sommeil, tongue tie, tongue chewing

## Principe central

- Langue sur le palais **24/7** — pas pousser, **suction / vide** (suction mew)
- Sans ça : maxillaire ne monte pas, mâchoire/pommettes reculent, airways étroites
- Effet cumulatif : temps total sur le palais compte (compounding)
- Langue ~600 g de force — 1,7 g suffit pour bouger une dent → expansion palais possible à tout âge (fascias, sutures, ostéoblastes)

## Suction mew (technique)

1. Dire « T » — langue au spot T
2. Sourire large + yeux ouverts + déglutition → nuque engage, posture monte, suction
3. Répéter déglutitions jusqu'à salive épuisée = vrai vide
4. **Semaine 1 rééducation** : toutes les heures, tenir la suction le plus longtemps possible

## Sommeil

- Même routine T + swallows avant coucher
- **Côté** (pas dos — gravité fait tomber la langue)
- Coussin entre genoux
- Tape zygomatique + mentalis → lip seal → mewing nocturne passif

## Tongue tie / force langue

- **Étirement frenulum** : T spot, pousser langue en arrière avec doigt 70 %, 30 s × 2 séries/j
- **Tongue chewing** : gomme naturelle (mastic), pousser contre palais avec pointe langue ~10 min/j

## Catalyseurs

- Thumb pull (technique correcte) + myofonction
- Alimentation dense, circadien, environnement — santé = looks

## 3 étapes produit

1. Apprendre + maintenir suction mew (~1 h/j rappel semaine 1–2)
2. Suction mew en dormant (côté + tapes)
3. Détendre fascia/langue + tongue chewing → genioglossus/styloglossus
