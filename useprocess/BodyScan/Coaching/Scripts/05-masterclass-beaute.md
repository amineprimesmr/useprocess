# Script 05 — Masterclass beauté (moche → attirant)

**Chaîne:** Enzo Geromegnace  
**Thème:** 3 piliers (visage, physique, posture) + charisme

## Vision centrale

- Beauté = **harmonie globale** visage + corps + posture
- Beaucoup ont un beau visage mais mauvaise posture → la beauté ne se voit pas
- Maîtriser les **3 piliers** = extrêmement rare → différenciation maximale
- Visage = reflet direct santé intérieure (hormones, énergie cellulaire)

## Pilier 1 — Visage

### Cortisol (ennemi #1)
- 95 % en stress chronique → cortisol trop élevé
- Détruit régénération cellulaire → acné, cernes, rides, cheveux fins/cassants, visage gonflé
- Mode survie = rétention d'eau faciale
- **Skincare chimique** = camouflage, empire à l'arrêt — régler la cause

### Solutions visage
| Action | Pourquoi |
|--------|----------|
| Baisser cortisol | Respiration nasale lente, parasympathique |
| Body fat ~13-15 % (homme) | Pas trop sec (<), pas >20 % |
| Calories denses 2500-3000 | Pas 1500 kcal carencées — relance sèche |
| Alimentation animaux/tubercules/miel/fruits | Carences = symptômes peau |
| Sommeil réparateur | Régénération nocturne |
| Rythme circadien | Lumière matin, repos soir, pas bleu/late sport/manger |
| Éradiquer huiles de graines | Première étape alimentation |
| Toxines environnement | Filtre douche, plastique/xénoestrogènes |
| Mâcher viande dure + mewing + déglution | Palais/maxillaire |
| Skincare naturel (karité, coco, suif) | Absorption cutanée = ingestion |

## Pilier 2 — Physique

- **Athlétique esthétique** : ni maigre, ni trop massif — fonctionnel, mobile, en V
- Détecter **déséquilibres** → commencer séance sur point faible, **amplitude**
- Pas 4×12 robotique → **périodisation concurrente** (changer toutes les 2-4 sem)
- Tableau progression (DPR) : battre perf précédente à charge égale
- Fonctionnel > machines (anneaux, unilatéral)
- Équilibre chaîne antérieure/postérieure, jambes, fessiers, cou

## Pilier 3 — Posture

- Bonne posture → langue sur palais **naturellement**
- Ex. SCM tendu → menton reculé → étirement ciblé
- Quads tendus → bassin antérieur → étirer quads, activer fessiers/ischios
- Mauvaise posture = pas charismatique, paraît faible/moche
- Visage extérieur (posture) + intérieur (hormones/énergie)

## Complément — Charisme & expression

- Parler, poser questions, lire à voix haute (pauses, débit, articulation)
- Éloquence = adapter au public
- Charisme = **contrôle** (environnement, voix, émotions, mouvements fluides)
- Démarche zen, pas mouvements brusques
