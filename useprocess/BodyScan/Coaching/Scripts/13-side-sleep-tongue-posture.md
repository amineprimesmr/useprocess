# Script 13 — Sommeil latéral + posture langue

**Thème:** dos = visage recule ; langue = moteur maxillaire ; sommeil côté = position fœtale

## Mécanisme

- Langue sur palais → support maxillaire → visage vers le haut/avant
- Dos : gravité, langue basse ~8 h/j → visage descend/recule
- Langue basse → fascia palais dense → ligne frontale superficielle (langue → pieds) → crâne restreint → posture cranienne
- Habitudes myofonction → parafonction : déglutition faible, mastication molle

## Solution

- Sommeil **sur le côté** (position fœtale, tribus) — airways + langue sur palais
- Cas : apnée, asthme, mâchoire récessée, dents crochues, asymétrie — corrigés par côté + langue

## Routine ce soir

1. Spot T + sourire max + **3 déglutitions** — maintenir position
2. Tape zyg joues + menton — fascias, rides, asymétrie, langue sur palais
3. Côté : coussin entre cuisses + **main sous tête**
4. Respiration **vers l'arrière dans le visage** (pas vers le crâne) — fascia thoracique + cervicale

## Message

- Nous sommes faits pour dormir sur le côté — déviation moderne
- Système nerveux = guérisseur — santé = looks
