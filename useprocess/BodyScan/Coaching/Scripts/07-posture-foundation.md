# Script 07 — Posture fondation (santé = looks)

**Thème:** posture globale, nuque, respiration, sommeil, pieds, chin tuck, face pulls, training fonctionnel

## Concepts clés

- Posture = fondation visage + corps — sans ça, produits et hacks inutiles
- Chaîne : pieds → bassin → dos → nuque → tête → maxillaire → respiration → CNS
- Nuque droite (pas chin tuck permanent) — assis, debout, marcher
- Respiration nasale ↔ posture (interliées)
- Sommeil latéral : 3 coussins (tête, entre genoux, à hugger) — airways ouvertes
- Marche / mouvement toutes les heures ; pieds nus ~30 min/jour
- Chin tuck : 2–3×12, maintien au fond — exercice temporaire, pas posture 24h
- Face pulls : 2–3×10–12 — rear delts, épaules, posture
- Sprint 8×15 s repos 1m30 ; training fonctionnel (système, pas isolation)
- Option : foam roller thoracique, dead hang lesté

## Ordre

1. Habitudes 24/7 (nuque, respiration, sommeil, marche, pieds)
2. Exercices correctifs (chin tuck, face pulls)
3. Training fonctionnel (sprints, chaînes de mouvement)
