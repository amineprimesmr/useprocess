# Script 06 — Génétique vs habitudes (santé = beauté)

**Chaîne:** Enzo Geromegnace  
**Thème:** pourquoi tu n'es pas "condamné" par la génétique — transformations Enzo, Luc, Saphir, Pacomè

## Message central

- **Pas la génétique** — habitudes quotidiennes → santé → attractivité
- Beauté = indicateur de santé (sélection biologique du partenaire sain)
- Amélioration possible à **tout âge** (20, 40, 60)
- Transformation en mois ; plein potentiel en **années** de bonnes habitudes
- Tu deviens **harmonieux**, tu ne choisis pas un type d'yeux précis

## Mécanismes détaillés

### Cortisol (approfondissement)
| Mode survie | Conséquence |
|-------------|-------------|
| Priorise cerveau, cœur, muscles | Peau/cheveux sans nutriments ni O₂ |
| 95 % stress chronique | Hormones déréglées (thyroïde négligée) |
| Regard peur | Sourcils hauts, front tendu — pas attrayant |
| Aldostérone élevée | Sodium → eau **extracellulaire** → visage bouffi + peau sèche |

### Toxines
- Antinutriments végétaux (goitrogènes, acide phytique) → carences + thyroïde
- Ultra-transformé, métaux lourds, microplastiques
- Xénoestrogènes (plastique chauffé) → hommes féminisés / déséquilibre femmes
- Corps détox = énergie volée à la régénération cellulaire

### Carences & macros
- 3 macronutriments nécessaires (pas carnivore exclusif)
- **Gras/cholestérol** → précurseur testo, DHT, progestérone, œstrogènes
- Micronutrition : produits animaux prioritaires
- Viande = construction tissus ; laitiers crus/œufs = régénération ; calcium/phosphore = os

### Sommeil
- Testostérone, DHT (5α-réductase), hormone de croissance, **mélatonine** produits la nuit
- Lumière bleue soir = mélatonine basse
- S'écrouler épuisé ≠ récupération (batterie vide)
- Pas dormir 4h — respecter rythme circadien

### Thyroïde T3/T4
| Profil | Mécanisme |
|--------|-----------|
| Fat / skinny fat | Hypothyroïdie probable — métabolisme bas |
| Skinny | Hyperthyroïdie possible — impossible prendre poids |
| Régimes 1000-1500 kcal | Tuent thyroïde et visage |

### Entraînement & posture
- **Concurrent training** (cycles 3-4 sem) + **biomécanique**
- Transfert force entre variantes — progresser sans stagnation 4×12
- Posture mauvaise = danger → sympathique → **cortisol**
- Posture bonne → langue palais → maxillaire projeté
