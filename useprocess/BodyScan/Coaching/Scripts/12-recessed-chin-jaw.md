# Script 12 — Menton récessif / mâchoire (orthotropics)

**Thème:** fascias → posture → muscles → os ; pas chirurgie — habitudes myofonction

## Chaîne

1. Fascias (22 kg, négligés) → posture → muscles corrects → adaptation osseuse (orthotropics / mechanotransduction)
2. Hyoïde / double menton = sling musculaire (digastrique, super-hyoïdiens, masseter, genioglossus/hyoglossus) — pas « naturel »

## Étape 1 — Sous-mâchoire / hyoïde

- **Neck curls** — active super-hyoïdiens
- **Souffle / expiration forcée** (type AeroPit) — digastrique, peau sous mâchoire
- **Suction mew** journée — sourire + yeux + déglutition, fascia activé

## Étape 2 — Mentalis

- Muscle mentalis hyperactif → menton fendu / witch chin
- **Tape kinésiologie en X sur menton** (STTO) chaque nuit — désactive mentalis, évite bruxisme
- Combiner tape zyg si besoin

## Étape 3 — Mastication mandibule

- Masseter > temporal — mordre, puis **rouler dents inférieures vers l'avant** lentement contre dents du fond
- Mécanotransduction — masseter = muscle le plus fort du corps
- Pas génétique — mauvaise mastication/déglution/posture depuis l'enfance

## Compléments

- Pression langue sur **spot T / papille incisive** ~30 min/j (tongue chewing ou pression pure)
- Maxilla antérieur lié mandibule → menton suit posture langue
- Soleil, alimentation dense — catalyseurs

## Résultats

- Illusion + adaptation progressive menton ; cas >25 ans documentés orthotropics
