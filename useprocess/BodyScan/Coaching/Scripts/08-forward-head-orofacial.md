# Script 08 — Tête en avant : cause orofaciale + correction structurelle

**Source:** Oscar Patel — forward head posture, myofunctional root cause

## Cause racine

- Tête en avant = compensation pour airways étroites (palais/maxillaire récessif)
- Habitudes orofaciales : mastication, déglution, respiration, posture langue
- Problème **structurel** avant musculaire — chin tucks seuls insuffisants sans myofonction

## Muscles (court terme)

- **Chin tuck avancé** : tête hors du lit/banc, 3×8–10, overload progressif — super-hyoïdiens, relâche trapèzes supérieurs
- **Nuque arrière** : sur le ventre, extension tête résistance mains/plaque — support crânien
- **APT** : RSS (release tennis ball, stretch hip flexors, strengthen glutes)
- **Marche** : orteils vers l'intérieur, talons vers l'extérieur, abdos légers — pas pieds en duck
- Pieds nus / chaussures barefoot

## Structure (long terme)

- **Langue** : tiers postérieur sur le palais — dents en contact léger
- **Platysma** : langue palais + pousser menton avant / gauche / droite 30 s
- **Déglutition** : sourire large + yeux hauts, 5 déglutitions
- **Orbicularis oris** : CPS, balloon hold si respiration obstruée
- **Thumb pull** : 8 semaines, palais + langue comme retainer
- **Conscience** : 2 semaines actives → suction mewing passif
- **Sommeil** : suction mewing, tape zyg, côté
- **Buteyko** : 3–4 min — sutures, fascias, posture nuque

## Séquence coaching

Fix muscles → fix structure squelettique → habitudes passives permanentes + posture corps (fascias pied → crâne).
