# Script 03 — Alimentation moderne vs alimentation dense (anti-gomuscu)

**Chaîne:** Enzo Geromegnace  
**Thème:** pourquoi l'alimentation fitness classique détruit santé + visage

## Concepts clés

### Problème — alimentation gomuscu
- Riz + poulet + brocolis + beurre cacahuète + compléments
- Conséquences : sèche/prise de masse ratées, acné, cernes, visage gonflé ou creusé, fatigue post-repas
- Corps en digestion permanente = zéro énergie = visage catastrophique

### Critique point par point
| Aliment | Verdict Enzo |
|---------|--------------|
| Riz | Pas optimal, arsenic, antinutriments — tubercules mieux |
| Poulet | OK mais peu dense — viande rouge + abats |
| Brocolis/légumes verts | Antinutriments (oxalates), pas naturel humain |
| Beurre cacahuète industriel | Huiles graines oxydées = vieillissement cellulaire |
| Compléments | Nutriments isolés sans cofacteurs, procédés toxiques |

### Solution — alimentation dense
- Produits animaux qualité (cru si possible) : viande rouge, abats, poissons gras, œufs (moins blanc)
- Laitiers **cru** (pas pasteurisé)
- Tubercules cuits vapeur (sans peau)
- Miel + fruits modérés (1 fruit/jour)
- Qualité : pâturage, sauvage, bio, pas pesticides

### Sèche / prise de masse
- **Sèche** : réduire laitiers crus quand plateau
- **Prise de masse** : augmenter laitiers crus / crème crue
- Plus de calories qu'en gomuscu MAIS densité nutritionnelle infiniment supérieure
- Corps en abondance, pas en famine → visage reste beau même en bulk

### Lien scan useprocess
- `skinClarityScore` bas → alimentation inflammatoire / carences
- `puffinessScore` / `underEyeFatigueScore` → digestion lourde, lymph, carences
- Relier au pilier 1 (hormones/thyroïde T3-T4, métabolisme)
